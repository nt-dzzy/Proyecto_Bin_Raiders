# Código Raspberry Pi – Bin Raiders

## Descripción
Scripts desarrollados en Python para la Raspberry Pi,
encargados de la lectura del sensor ultrasónico y el
envio de datos a la aplicación Android.

## Funcionalidades
- Lectura de distancia del sensor ultrasónico
- Simulación de medición
- Captura de imagen mediante cámara
- Envío de datos (local)

## Requisitos
- Raspberry Pi 4
- Python 3
- GrovePi (sensor ultrasónico)
